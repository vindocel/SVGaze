## FAQ

### General
- **Q: Is SVGaze free?**
  - A: Yes! SVGaze is 100% free and open-source under MIT license.

- **Q: Do my files get uploaded anywhere?**
  - A: No! Everything runs locally in your browser. Your files never leave your device.

- **Q: What browsers are supported?**
  - A: Chrome 90+, Edge 90+, Safari 14+. Firefox 88+ (experimental).

- **Q: Can I use SVGaze offline?**
  - A: Yes, if you run it locally. The online version requires internet.

### Features
- **Q: How many SVGs can I view at once?**
  - A: Thousands! The gallery is optimized for large icon libraries.

- **Q: Can I edit SVG code directly?**
  - A: Yes! The editor has full syntax highlighting and validation.

- **Q: What export formats are supported?**
  - A: Preview, React, React Native, PNG, and Data URI.

### Troubleshooting
- **Q: Why can't I select a folder?**
  - A: You need to use a modern browser with File System Access API support.

- **Q: Icons appear black in dark theme?**
  - A: Check if your SVGs have hardcoded fill colors. Use the color picker to override.

- **Q: Search not working?**
  - A: Make sure JavaScript is enabled and check browser console for errors.
